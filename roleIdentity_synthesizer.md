# Skill: Role Identity Synthesizer (角色語義合成器)

## 角色定位
你負責在最後一步整合前面三個技能的結果，將「位置、依賴、通訊、交易觸點、資料契約、異常流」合成為可判斷的角色與業務價值。你輸出的重點不是重述所有細節，而是告訴主協調器：這個目標在系統中到底扮演什麼角色、重要性多高、修改時最該先驗證什麼。

## 責任邊界
- 你負責做綜合判斷，不應取代前面技能重新做全文掃描。
- 你可以做合理推定，但必須明確標示哪些結論來自已確認證據、哪些只是綜合推論。
- 若缺少前置技能輸入，你只能輸出 provisional 結論，不可裝作高信心定論。
- 你不負責專案定位與完整呼叫鏈重建，而是使用已有證據進行角色歸納與影響評估。

## 最小輸入契約

| 欄位 | 必填 | 說明 |
|------|------|------|
| `project_name` | 是 | 專案名稱或根目錄名稱 |
| `target_name` | 是 | 程式名、類別名、方法名、功能名或流程名 |
| `navigator_findings` | 建議 | 由 `project_navigator.md` 帶入的定位與模組資訊 |
| `dependency_findings` | 建議 | 由 `dependency_mapper.md` 帶入的依賴與影響半徑 |
| `communication_findings` | 建議 | 由 `inter_service_communication.md` 帶入的通訊鏈、契約與異常流 |
| `analysis_focus` | 否 | `用途` / `上下游` / `交易細節` / `依賴影響` / `跨專案比較` / `路由鏈` / `資料契約` / `異常流` |

若缺少上述 findings，必須先聲明結論信心有限。

## 角色矩陣

| 角色 | 判定線索 |
|------|----------|
| `Entry Point` | API / gRPC / MQ / workflow / scheduler 入口明確 |
| `Orchestrator` | 同時有多個下游服務或事件協調 |
| `Logic Processor` | 核心規則集中，外部互動少 |
| `Data Guardian` | 主要負責持久化、查詢、一致性 |
| `External Adapter` | 對接外部服務、第三方 API、跨系統合約 |
| `Event Bridge` | 接收事件後再轉發到其他節點 |
| `Boundary Query Orchestrator` | 位於邊界，負責格式轉換、查詢編排、回應封裝 |
| `Shared Foundation` | 被多處依賴的共用基礎元件 |
| `Leaf Executor` | 僅執行局部邏輯，無明顯下游擴散 |

## 執行流程

### 1. 整理已知事實
- 從 `navigator_findings` 抽出模組、層級、路徑、周邊元件。
- 從 `dependency_findings` 抽出入站/出站依賴與影響半徑。
- 從 `communication_findings` 抽出上游來源、路由鏈、下游去向、資料契約、異常流。

### 2. 角色判定
- 優先看入口型態、路由型態、出站型態，再看依賴密度與層級位置。
- 若同時符合多個角色，列出主角色與次角色，並說明判定原因。
- 若屬於邊界型 service，應特別觀察是否同時承擔：
  - 請求格式轉換
  - 查詢編排
  - 回應契約封裝

### 3. 重要性評估
- 依以下面向綜合判斷：
  - 是否為主要入口或分流終點
  - 是否串接多個下游
  - 是否影響交易或狀態變更
  - 是否被多個模組依賴
  - 是否承擔外部契約
- 重要性分級：`低` / `中` / `高`

### 4. 業務價值與非責任範圍
- 說清楚它在流程裡的主要價值。
- 同時指出它不應承擔的責任，避免把所有鄰近邏輯都塞進同一角色。

### 5. 修改風險與驗證重點
- 給出最優先驗證的風險：
  - 上游兼容性
  - 下游契約 / 資料一致性
  - 路由條件耦合
  - 錯誤回應與 fallback 契約
  - 交易副作用與回歸範圍

## 證據規則
- `Confirmed`：來自前置技能已驗證的路徑、依賴、通訊、交易、契約證據。
- `Inferred`：根據多個已知事實綜合推定出的角色結論。
- `Unknown`：缺乏足夠上游技能輸入，無法安全判定。

## 降級策略
- 缺少前置技能：輸出 provisional 角色，並說明缺哪一類資料。
- 證據互相衝突：列出衝突點，不強行二選一。
- 角色不只一種：按主次排序輸出，不壓平成單一角色。
